# 📯 Локальная игра «Домино» (FastAPI + WebSocket)

Привет! Это подробная инструкция, как скачать, настроить и запустить игру у себя на компьютере. Просто делай всё по порядку, и всё заработает.

---

## Шаг 1. Установка Python (если его еще нет)

Для работы игры на компьютере должен быть установлен Python версии 3.10 или новее.

1. Перейди на официальный сайт [python.org/downloads](https://www.python.org/downloads/).
2. Нажми большую желтую кнопку **Download Python 3.x.x**.
3. **ВАЖНО:** Когда запустишь скачанный установщик, в самом первом окне (внизу) **обязательно поставь галочку "Add Python to PATH"**. Без этого ничего не заработает!
4. Нажми **Install Now** и дождись окончания.

---

## Шаг 2. Скачивание проекта из GitHub

Выбери **один** из двух способов, который тебе удобнее:

* **Способ А (Если умеешь пользоваться Git):**
    Открой терминал/консоль и выполни команду:
    ```bash
    git clone <ССЫЛКА_НА_ТВОЙ_РЕПОЗИТОРИЙ>
    ```
* **Способ Б (Самый простой):**
    1. На странице этого репозитория на GitHub нажмите на зеленую кнопку **Code** (сверху справа).
    2. В выпадающем меню выбери **Download ZIP**.
    3. Распакуй скачанный архив в любое удобное место (например, на Рабочий стол).

---

## Шаг 3. Установка библиотек

Теперь нужно установить модули, на которых написан сервер.

1. Открой командную строку (В Windows: нажми `Win + R`, введи `cmd` и нажми Enter).
2. Нам нужно зайти в папку с проектом. Для этого введи команду `cd ` (с пробелом на конце), а затем **перетащи папку с проектом мышкой прямо в окно консоли**. Путь подставится сам. Нажми Enter.
   *Пример команды:* `cd C:\Users\User\Desktop\domino-project`
3. Скопируй и выполни следующую команду для установки библиотек:
   ```bash
   python -m pip install fastapi "uvicorn[standard]"

```

4. Дождись окончания загрузки. Если нет красных надписей со словом `ERROR` — всё прошло успешно.

---

## Шаг 4. Запуск сервера игры

1. В том же окне консоли (где ты уже находишься в папке проекта) введи команду запуска:
```bash
uvicorn main:app --reload --port 8000

```


2. В консоли побегут строчки, и появится надпись: `Uvicorn running on http://127.0.0.1:8000`.
3. **ВАЖНО:** Не закрывай это черное окно консоли! Пока оно открыто — сервер игры работает. Если закроешь — игра отключится.

---

## Шаг 5. Как начать играть (Инструкция для двоих)

Так как игра сетевая и работает через WebSockets, играть нужно в два окна.

1. Открой любой браузер и переходи по адресу: **`http://localhost:8000`**
2. Ты увидишь окно входа. В поле **Room ID** введи любое число (например, `1`) и нажми кнопку **Join**. Экран напишет, что ждет оппонента.
3. Теперь сделай **одно** из двух:
* **На одном ПК:** Открой *новую вкладку* браузера (или другой браузер, например Edge, если до этого был Chrome).
* **В локальной сети (с другого устройства):** Если твой друг подключен к тому же Wi-Fi, он может зайти со своего ноутбука или телефона. Только вместо `localhost` ему нужно будет ввести твой локальный IP-адрес (например, `http://192.168.1.50:8000`).


4. Второй игрок вводит **ТОТ ЖЕ САМЫЙ** Room ID (число `1`) и нажимает **Join**.
5. Как только оба подключатся, интерфейс обновится, вам раздадут по 7 фишек и начнется игра!

---

## 🎮 Коротко о правилах в интерфейсе

* Игра сама следит за ходами. Сверху всегда написано, чей сейчас ход (Твой или Оппонента).
* Чтобы сделать ход, просто кликни по фишке в своей руке.
* Система сама проверит, подходит ли фишка к левому или правому краю цепочки на столе, и автоматически выставит её.
* Если фишка не подходит, система автоматически возьмет фишку из колоды (базара). Если и в колоде ничего нет, ход автоматически передастся другому игроку.
* Игра идет до тех пор, пока у кого-то полностью не закончатся фишки на руках.

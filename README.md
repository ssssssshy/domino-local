# Domino — локальная веб-игра домино

Простой серверный проект на FastAPI + WebSocket для игры в домино локально между двумя игроками.

Содержимое репозитория
- main.py — сервер (FastAPI + WebSocket)
- templates/index.html — фронтенд страница
- static/app.js — клиентская логика WebSocket и UI
- static/style.css — минимальные стили

Требования
- Python 3.10+
- Рекомендуемые пакеты: fastapi, uvicorn[standard]
  (или fastapi + websockets)

Установка зависимостей
```bash
python -m pip install --user fastapi "uvicorn[standard]"
# или, если хотите только websockets:
# python -m pip install --user fastapi websockets
```

Запуск сервера
```bash
# из папки проекта (там где main.py)
uvicorn main:app --reload --port 8000
```
Откройте браузер и перейдите по адресу: http://localhost:8000

Как играть
1. Откройте сайт в двух вкладках/браузерах (или на двух машинах в локальной сети).
2. Введите одинаковый идентификатор комнаты (например: 1) и нажмите Join.
3. При подключении двух игроков автоматически раздаётся по 7 фишек, начинается игра.
4. Нажмите на кнопку с фишкой в вашей руке, чтобы попытаться её выложить. Правила:
   - Фишка может быть выложена слева или справа от цепочки, если её сторона совпадает с торцом.
   - Если не подходит — автоматически берётся фишка из колоды (если остались). Если вытянутая подходит, она автоматически выкладывается.
   - Если колода пуста и ход невозможен — ход пропускается.
   - Побеждает игрок, у которого кончились фишки.
